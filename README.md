# GUI-based-chat-application
GUI based chat application in python

APPLICATIONS:
  Register,
  Login,
  Forgot password-reset password by sending otp to email,
  check users online,
  send messages,
  send files,
  delete chat history.
  
LIBRARIES USED:
  socket,
  threading,
  tkinter,
  random,
  string,
  smtplib,
  email.
